### CSV2MYSQL Script
#### This script converts the Big-Vul CSV Dataset to a MySQL Table
---
> How to
- Create a pip environment `python -m venv env`
- Enter the pip environment `source env/bin/activate`
- Install the dependencies `pip install -r requirements.txt`
- Create the destiny database on MySQL
- Set your MySQL authentication details in the `auth` file
- Run the `csvtomysql.py` file `python csvtomysql.py`
